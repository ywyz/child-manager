#### Q5：自然周起始日

### 3.3 数据模型、保留与任务关系

- **状态**：必须同步。
- **阻断 M0**：是。

#### Q8：AI 结果清理与一年审计

- **结论**：清理完整 `output_content` 和不再需要的详细输入，不删除整条结果记录、任务身份或审计关联。
- **保留内容**：`job_id`、状态、模型与提示词版本、Schema、`output_sha256`、必要操作者和时间元数据。
- **隐私边界**：审计至少保留一年，但不得复制完整教案或 AI 输入输出。
- **状态**：必须同步。
- **处理方式**：随 Q22 一并处理，不单独新增重复设计。
- **阻断 M0**：是，由 Q22 统一关闭。

#### Q9：后台任务父子与重试外键

- **结论**：增加可空同园组合自外键 `(kindergarten_id, parent_job_id) -> background_jobs(kindergarten_id, id)`，删除策略为 `RESTRICT`。
- **关系区分**：`parent_job_id` 只表达 `ai.batch` 父子关系；`retry_of_job_id` 使用独立的可空同园组合自外键表达显式重试谱系。
- **约束**：禁止自引用；以 `(kindergarten_id, parent_job_id, target_section)` 唯一约束防止同一批次重复栏目子任务。

- **阻断 M0**：是。

### 3.4 加密与密钥

#### Q10：AI Key 加密算法

- **结论**：使用 AES-256-GCM；每次加密生成随机 96 位 nonce，AAD 绑定园所 ID、模型档案 ID 和 envelope version。
- **轮换**：通过 `key_id` 支持新密钥写入、旧密钥只读、分批重加密和验证后移除。
- **失败边界**：认证标签验证失败必须拒绝解密，不得返回部分明文。
- **状态**：必须同步。
- **阻断 M0**：是。

#### Q11：主加密密钥来源

- **结论**：开发环境从仓库外、受文件权限保护的本地密钥文件读取；自动化测试使用明确的测试适配器。
- **禁止项**：主密钥不得进入代码库、数据库、镜像、普通环境变量、日志、异常、审计或测试快照。
- **延后项**：生产密钥托管方案延后至 M9。
- **修改目标**：`AGENTS.md`、系统架构和相关 Spec Kit 任务；不把运行时密钥来源细节塞进领域数据模型。
- **状态**：必须同步。
- **阻断 M0**：是。

### 3.5 API、任务状态与幂等

#### Q12：任务状态机缺少 `expired`

- **结论**：补充 `awaiting_confirmation -> expired`。`expired` 是等待教师确认的 AI 预览业务终态。
- **副作用**：设置 `finished_at`；不得再采用或拒绝；不修改教案正文，不创建快照。
- **适用范围**：`ai.batch` 父任务状态仍由四个子任务派生；Word 导出和无需确认的任务不进入 `expired`。
- **状态**：必须同步。
- **阻断 M0**：是。

#### Q13：幂等键定义

- **结论**：客户端幂等 scope 为园所、操作者、HTTP 方法、规范化路由模板和 key；fingerprint 另含规范化实际 path、业务 query 和 canonical JSON body。
- **行为**：相同 scope、key 和 fingerprint 返回原结果；相同 scope/key 但 fingerprint 不同返回冲突。
- **修改目标**：系统架构应引用并概括 Spec Kit/OpenAPI 契约，不复制易漂移的完整字段清单。
- **状态**：必须同步。
- **阻断 M0**：是。

### 3.8 编辑与 AI 采用

#### Q19：新增集体活动环节的插入位置

- **结论**：AI 返回单一结构化 `step` 和 `suggested_insert_index`，合法范围为 `0 <= index <= len(process)`。
- **校验**：越界属于 AI 结构错误并进入既定自动重试，不得截断、改成末尾或自行猜测。
- **采用**：结果先作为预览；教师明确采用且当前版本仍与生成基线一致时，系统按有效索引插入并设置 `is_ai_added=true`。内容已变化时返回冲突。
- **状态**：已满足。
- **阻断 M0**：否。

#### Q20：自动保存失败

- **结论**：不进行后台自动重试；保留页面内容并显示失败状态，允许用户手动重试。继续编辑可以触发下一次正常自动保存。
- **错误边界**：版本冲突、权限错误和校验错误不得通过隐藏重试绕过。
- **状态**：已满足。
- **处理方式**：任务清单只需核对失败行为测试覆盖。
- **阻断 M0**：否。
